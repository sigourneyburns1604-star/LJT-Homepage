# Abstract
This work proposes a method to compose parameter-efficient modules using arithmetic operations, enabling efficient combination of multiple fine-tuned adapters in large language models.

## Authors
Jinghan Zhang, Shiqi Chen, Junteng Liu (co-author), Junxian He
